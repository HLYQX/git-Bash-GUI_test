1.To quit the git log, enter a "q" in git bash.
2.To create an ".gitignore" in a repository, enter "touch .gitignore" in git bash.
3.To rebase the commits, use the git gui to know the commit ID, and then enter "git rebase -i [commitID]" in git bash, enter "i" to insert "p" or "s", press “esc”and enter ":wq" to save and quit, then edit the commit messages and enter ":wq" again to rebase.
4.Use "git rebase --abort" to abort the rebase.
